# MBA IA Generativa - Python

Repositorio da disciplina com exercicios e materiais de apoio organizados por aula.

## Estrutura

- `PYAI/aula01/` - exercicios introdutorios de Python.
- `PYAI/aula02/` - exercicios da segunda aula.
- `PYAI/aula03/` - estrutura e documentacao da terceira aula sobre modularizacao e orientacao a objetos.
- `PYAI/aula04/` - materiais e exemplos da quarta aula sobre orientacao a objetos parte 2.
- `PYAI/aula05/` - exemplos praticos da quinta aula com consumo e criacao de APIs.
- `PYAI/aula06/` - scaffold base da sexta aula, com ambiente e workspace prontos para novos conteudos.

## Como executar

Use Python 3 no terminal:

```bash
python3 PYAI/aula01/ex01_imc.py
python3 PYAI/aula01/ex02_circulo.py
python3 PYAI/aula01/ex03_bhaskara.py
```

Para novas aulas, adicione arquivos na pasta correspondente.
