# MBA IA Generativa - Exercicios de Python

Este repositorio contem exercicios introdutorios de Python desenvolvidos na primeira aula.

## Exercicios

1. `imc.py` - calculo de IMC a partir de peso e altura.
2. `circulo.py` - calculo de perimetro e area de um circulo.
3. `bhaskara.py` - calculo das raizes reais de uma equacao de segundo grau.

## Como executar

Use Python 3 no terminal:

```bash
python3 imc.py
python3 circulo.py
python3 bhaskara.py
```
