# MBA IA Generativa - Exercicios de Python

Este repositorio contem exercicios introdutorios de Python desenvolvidos na primeira aula.

## Exercicios

1. `aula01/imc.py` - calculo de IMC a partir de peso e altura.
2. `aula01/circulo.py` - calculo de perimetro e area de um circulo.
3. `aula01/bhaskara.py` - calculo das raizes reais de uma equacao de segundo grau.

## Como executar

Use Python 3 no terminal:

```bash
python3 aula01/imc.py
python3 aula01/circulo.py
python3 aula01/bhaskara.py
```
