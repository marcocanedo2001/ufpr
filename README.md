# MBA IA Generativa - Exercicios de Python

Este repositorio contem exercicios introdutorios de Python desenvolvidos na primeira aula.

## Exercicios

1. `aula01/ex01_imc.py` - calculo de IMC a partir de peso e altura.
2. `aula01/ex02_circulo.py` - calculo de perimetro e area de um circulo.
3. `aula01/ex03_bhaskara.py` - calculo das raizes reais de uma equacao de segundo grau.

## Como executar

Use Python 3 no terminal:

```bash
python3 aula01/ex01_imc.py
python3 aula01/ex02_circulo.py
python3 aula01/ex03_bhaskara.py
```

## Trabalho em equipe

- Crie uma branch por tarefa, por exemplo: `feat/aula01-ex01`.
- Abra Pull Request para revisao antes de mergear na `main`.
- Siga o guia em `CONTRIBUTING.md`.
